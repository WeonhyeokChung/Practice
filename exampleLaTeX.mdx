Type I error: Reject null when the true state is null. 

Type II error: Not reject null when the true state is altenative. 

Two hypothesis:

$$
H_0 : \mu_0 = \mu_1
$$
$$
H_1 : \mu_0 \ne \mu_1
$$
Let
$$
E[\bar{Y_1}-\bar{Y_0}] = \mu_1 - \mu_0
$$
